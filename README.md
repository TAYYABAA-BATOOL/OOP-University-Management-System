# OOP-University-Management-System

A clean implementation of Object-Oriented Programming (OOP) principles using PHP. This project demonstrates core software engineering concepts required for maintainable and scalable code.

### 🔑 Key OOP Concepts Implemented:
- **Encapsulation**: Used private properties with public getters to secure data.
- **Inheritance**: `Student` and `Teacher` classes inherit common properties from the `Person` base class.
- **Polymorphism**: `getDetails()` method behaves differently for different objects.

### 🛠️ How to run:
1. Ensure PHP is installed.
2. Run command: `php index.php`
